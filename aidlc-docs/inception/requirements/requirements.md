## 1. プロジェクト概要

### ゴール
自身のスキルと実績をアピールできる、GitHub Pages でホスティングされる日本語ポートフォリオサイトを構築する。
インターン応募の選考ツールとして機能しつつ、長期的にキャリア全体を通じて育て続けられる資産とする。

### ターゲットユーザー
| 優先度 | ユーザー | 目的 |
|--------|---------|------|
| Primary | インターン採用担当者 | 候補者のスキル・実績を短時間で判断する |
| Secondary | 技術コミュニティ / 同業者 | エンジニアとしての技術的深さを確認する |
| Future | フリーランス案件クライアント | 実績と連絡方法を確認する |

## 2. 機能要件

### 2.1 MVP スコープ（v1.0）

#### FR-01: Hero / 自己紹介セクション
- 名前・キャッチコピー（自分を一言で表すフレーズ）を表示する
- プロフィール写真またはアバター画像を表示する
- 「詳しく見る」スクロールナビゲーションを設ける
- ページ読み込み時にアニメーションを表示する

#### FR-02: スキルセットセクション
- 技術スタック（言語、フレームワーク、ツール）を視覚的に一覧表示する
- カテゴリ別（例: Frontend / Game / Infrastructure）に分類する
- 習熟度を視覚的に表現する（バー、アイコン、ラベル等）

#### FR-03: 制作物・実績セクション
- プロジェクトカードを一覧表示する（タイトル、説明、使用技術、リンク）
- GitHub リポジトリ / デモサイトへのリンクを設ける
- 画像またはサムネイルを表示する
- カード展開 or モーダルで詳細説明を表示できる

#### FR-04: GitHub / SNS リンクセクション
- GitHub, Twitter/X, Zenn 等の外部プロフィールへのリンクを表示する
- フッターまたは専用セクションとして配置する

### 2.2 Phase 2 スコープ（v1.1）

#### FR-05: 職歴・学歴セクション
- 学歴・インターン経験等をタイムライン形式で表示する

#### FR-06: ブログ・記事リンクセクション
- Zenn / Qiita 等の外部記事を一覧表示する
- 外部 API またはリンク集として実装する

### 2.3 Future スコープ（v2.0）

#### FR-07: WebGL ゲーム埋め込み
- Unity / Godot の WebGL エクスポートをページ内に iframe または直接埋め込む
- ゲームごとに個別ページ（ルート）を設ける
- ゲーム一覧ページからナビゲートできる

## 3. 非機能要件

### 3.1 パフォーマンス
- Lighthouse Performance スコア 90 以上（GitHub Pages ホスティング環境）
- 初回ページ読み込み: LCP 2.5 秒以内
- アニメーションは `prefers-reduced-motion` に対応する

### 3.2 アクセシビリティ
- Lighthouse Accessibility スコア 90 以上
- 画像に alt テキストを設定する
- キーボードナビゲーションに対応する

### 3.3 レスポンシブ対応
- モバイル（320px〜）/ タブレット / デスクトップ（1440px+）に対応する
- スマートフォンからの閲覧を想定したレイアウトを優先する

### 3.4 保守性
- プロジェクトデータを JSON / MDX ファイルで管理し、コンポーネントを変更せずに実績追加できる構成にする
- 新しい実績追加はファイル1つの追加のみで完結させる

### 3.5 セキュリティ・プライバシー
- 詳細な個人情報（住所、電話番号）は掲載しない
- 連絡先は SNS / GitHub 経由とする（フォームは v1 スコープ外）
- 使用フォント・ライブラリはすべてライセンスが明確なものを選定する

## 4. 技術制約

| 項目 | 内容 |
|------|------|
| ホスティング | GitHub Pages（静的ファイルのみ） |
| フレームワーク | Next.js 15+（`output: 'export'` 静的エクスポート） |
| 言語 | TypeScript |
| サイト言語 | 日本語のみ（v1） |
| バックエンド | なし（サーバーサイド処理不可） |
| データストア | JSON / MDX ファイル（ローカル） |
| お問い合わせ | v1 はスコープ外（将来的に Formspree 等を検討） |

---

## 5. スコープ外（v1.0）

- お問い合わせフォーム
- 多言語対応（英語）
- CMS 連携
- WebGL ゲーム埋め込み
- ダークモード切り替え（デザイン実装次第で検討）
- アクセス解析（Google Analytics 等）

## 6. リリース基準（MVP）

- [ ] Hero / 自己紹介セクションが表示される
- [ ] スキルセットセクションが表示される
- [ ] 制作物・実績が 1 件以上表示される
- [ ] GitHub プロフィールへのリンクが機能する
- [ ] モバイル・デスクトップで正常に表示される
- [ ] GitHub Pages への自動デプロイが設定されている
- [ ] Lighthouse Performance / Accessibility 各 90 以上
